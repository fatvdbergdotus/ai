# Data Architectures

| Concept            | Type / Category              | Main Purpose                                      | Data Type Supported                         | Key Characteristics                                                                 | Typical Use Cases                              |
|-------------------|----------------------------|--------------------------------------------------|--------------------------------------------|--------------------------------------------------------------------------------------|-----------------------------------------------|
| Data Fabric        | Architecture (integration) | Unified access across distributed data systems   | All (structured, semi, unstructured)        | Metadata-driven, automated integration, connects multiple environments               | Enterprise data integration, analytics access  |
| Data Lake          | Storage system             | Store raw data in original format                | All types                                   | Schema-on-read, highly scalable, flexible storage                                    | Big data, ML, unstructured data processing     |
| Data Mesh          | Architectural approach     | Decentralized data ownership                     | All types                                   | Domain-driven, data as a product, self-service infrastructure                        | Large organizations, scalable data platforms   |
| Data Warehouse     | Storage & analytics system | Store processed, structured data for analysis    | Structured                                  | Schema-on-write, optimized for queries, high performance for BI                      | Reporting, dashboards, business intelligence   |
| NoSQL Database     | Database system            | Flexible, scalable data storage                  | Semi-structured & unstructured              | No fixed schema, distributed, high scalability                                       | Real-time apps, big data, distributed systems  |
| OLAP Cube          | Analytical structure       | Fast multidimensional data analysis              | Structured                                  | Pre-aggregated data, multidimensional (time, region, product)                        | Business analysis, slicing & dicing data       |
